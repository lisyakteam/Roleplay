## Features
- <b>Player leashing</b> (similar to mobs)
- <b>Player interactions</b> (pat by pressing shift + RMB)
- <b>Gift crafting</b> (including pranks!)

Player leashing disables player collision. You may disable leashing in config if you don't like that!

> [!WARNING]
> Leads and gift crafting may cause item duplication due to insufficient testing!
> If you find any bug, please open [an issue](https://github.com/lisyakteam/Roleplay/issues).
